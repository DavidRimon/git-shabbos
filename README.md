# git-shabbos

## Yeddishkite for git!
Change your favorite git commands!

## Examples:
`git shabbos` = `git push`

`git vach` = `git pull`

`git yomtev` = `git checkout`

And more!

## Install

Simply run the `git-alias.sh` script
